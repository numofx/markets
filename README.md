# App

A web application to interact with a short term money market on Celo for ERC-20 based KES and USD stablecoins. Users can borrow or lend FX at a fixed rate for a short duration, 3 months. 

### Usage

Make sure to run `bun install` first!

| Command       | Description              |
| ------------- | ------------------------ |
| `just dev`    | Start development server |
| `just build`  | Build for production     |
| `just start`  | Start production server  |
| `just clean`  | Clean build artifacts    |
| `just deploy` | Deploy to Vercel         |
